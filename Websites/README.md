### 1. ğŸ“š Hobi Ngoding (Educational Portal)
A responsive landing page designed to encourage new students to learn web development.
* **Tech Stack:** HTML5, CSS3, FontAwesome, Google Fonts.
* **Key Features:**
    * **Responsive Navbar:** Collapses into a hamburger menu on mobile.
    * **Pulse Animation:** CSS animations on the "Start Learning" button.
    * **Topic Grid:** Layouts for HTML, CSS, JS, and PHP course cards.

### 2. ğŸ•¹ï¸ Bouncing Ball (Canvas Physics)
A browser-based physics experiment using the HTML5 Canvas API.
* **Tech Stack:** HTML5 Canvas, JavaScript, Glassmorphism CSS.
* **Mechanics:**
    * **Physics Engine:** Calculates velocity, friction, and wall collisions.
    * **Interactive Controls:** Click the screen to boost speed or shake the ball.
    * **Dynamic Colors:** The ball changes color every time it hits a wall.

### 3. ğŸ¤ª Cosmic Alarm Setter (The "Worst UX" Experiment)
A satirical project designed to break every rule of User Experience (UX). Instead of typing the time, you must drag the entire earth.
* **Tech Stack:** Three.js (3D Library), Tailwind CSS.
* **The "Bad" Features:**
    * **3D Time Control:** You must drag the 3D Earth to change the time (1 pixel drag = 30 minutes).
    * **Runaway Button:** The "Set Alarm" button physically runs away from your mouse cursor.
    * **Unhelpful UI:** Instructions are vague ("Time is relative"), and the cursor disappears.
