Materi Dasar PHP

Pelajari bahasa pemrograman server-side untuk membangun aplikasi web dinamis.

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk membuat website dinamis. Berbeda dengan HTML, CSS, dan JavaScript yang berjalan di browser (client-side), PHP dijalankan di server dan hasilnya dikirim ke browser sebagai HTML.

üí° Tahukah Anda?
PHP digunakan oleh lebih dari 77% website di internet, termasuk Facebook, Wikipedia, dan WordPress!

Keunggulan PHP:

  • Gratis dan open-source
  • Mudah dipelajari untuk pemula
  • Terintegrasi baik dengan database (MySQL, PostgreSQL)
  • Cross-platform (Windows, Linux, macOS)
  • Komunitas besar dan dokumentasi lengkap

Sintaks Dasar PHP

Kode PHP ditulis di antara tag <?php dan ?>

<?php
// Ini adalah komentar satu baris

/* Ini adalah
   komentar multi baris */

// Menampilkan teks
echo "Hello, World!";

// Atau menggunakan print
print "Selamat datang di PHP!";
?>

Output:

Hello, World!
Selamat datang di PHP!

Variabel & Tipe Data

Variabel di PHP diawali dengan tanda $ dan bersifat case-sensitive.

<?php
// Variabel
$nama = "Budi";
$umur = 25;
$tinggi = 175.5;
$aktif = true;

// Menampilkan variabel
echo "Nama: " . $nama . "<br>";
echo "Umur: $umur tahun<br>";

// Tipe data
$angka = 100;           // Integer
$desimal = 10.5;        // Float
$teks = "Hello";        // String
$benar = true;          // Boolean
$kosong = null;         // NULL
$array = [1, 2, 3];     // Array
?>

Output:

Nama: Budi
Umur: 25 tahun

Operator

PHP memiliki berbagai operator untuk melakukan operasi.

<?php
// Operator Aritmatika
$a = 10;
$b = 5;
echo $a + $b;  // 15

// Operator Perbandingan
$x = 10;
$y = "10";
var_dump($x == $y);   // true (nilai sama)
var_dump($x === $y);  // false (nilai dan tipe berbeda)

// Operator Logika
var_dump(true && false);  // false (AND)

// Operator String
$nama_depan = "Budi";
$nama_belakang = "Santoso";
$nama_lengkap = $nama_depan . " " . $nama_belakang; // Budi Santoso
?>

Kondisi (If-Else)

Digunakan untuk membuat keputusan berdasarkan kondisi tertentu.

<?php
$nilai = 85;

if ($nilai >= 90) {
    echo "Grade: A";
} elseif ($nilai >= 80) {
    echo "Grade: B";
} else {
    echo "Grade: D";
}
?>

Output:

Grade: B

Perulangan (Loop)

Digunakan untuk mengulang blok kode beberapa kali.

<?php
// For Loop
for ($i = 1; $i <= 3; $i++) {
    echo "Iterasi ke-$i<br>";
}

// Foreach (untuk array)
$buah = ["Apel", "Jeruk", "Mangga"];
foreach ($buah as $item) {
    echo "- $item<br>";
}
?>

Output:

Iterasi ke-1
Iterasi ke-2
Iterasi ke-3

- Apel
- Jeruk
- Mangga

Form Handling

PHP sangat baik dalam memproses data dari form HTML.

<!-- HTML Form -->
<form action="proses.php" method="POST">
    <input type="text" name="nama">
    <button type="submit">Kirim</button>
</form>

<!-- proses.php -->
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nama = htmlspecialchars($_POST['nama']);
    if (empty($nama)) {
        echo "Nama harus diisi!";
    } else {
        echo "Terima kasih, $nama!";
    }
}
?>

Koneksi Database (MySQL)

PHP dapat terhubung dengan database untuk menyimpan dan mengambil data.

<?php
// 1. Koneksi ke Database (MySQLi)
$conn = mysqli_connect("localhost", "root", "", "belajar_php");
if (!$conn) { die("Koneksi gagal: " . mysqli_connect_error()); }

// 2. SELECT - Mengambil data
$sql = "SELECT * FROM users";
$result = mysqli_query($conn, $sql);

while ($row = mysqli_fetch_assoc($result)) {
    echo $row['nama'] . " - " . $row['email'] . "<br>";
}

mysqli_close($conn);
?>
