Materi Dasar JavaScript

Pelajari bahasa pemrograman untuk membuat website interaktif dan dinamis.

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Dengan JavaScript, Anda dapat merespons aksi pengguna, memanipulasi HTML dan CSS, serta membuat aplikasi web yang dinamis.

Variabel & Tipe Data

Variabel digunakan untuk menyimpan data. Di JavaScript, gunakan let, const, atau var untuk mendeklarasikan variabel.

// Mendeklarasikan variabel
let nama = "Budi";
const umur = 25;
let isActive = true;

// Tipe data
let angka = 100;           // Number
let teks = "Hello";        // String
let benar = true;          // Boolean
let kosong = null;         // Null
let tidakTerdefinisi;      // Undefined

Operator

Operator digunakan untuk melakukan operasi pada variabel dan nilai.

// Operator Aritmatika
let a = 10;
let b = 5;
let tambah = a + b;    // 15
let kurang = a - b;    // 5
let kali = a * b;      // 50
let bagi = a / b;      // 2

// Operator Perbandingan
10 == "10"   // true (nilai sama)
10 === "10"  // false (nilai dan tipe berbeda)
10 != 5      // true
10 > 5       // true

// Operator Logika
true && false  // false (AND)
true || false  // true (OR)
!true          // false (NOT)

Kondisi (If-Else)

Digunakan untuk membuat keputusan berdasarkan kondisi tertentu.

let nilai = 85;

if (nilai >= 90) {
    console.log("Grade: A");
} else if (nilai >= 80) {
    console.log("Grade: B");
} else if (nilai >= 70) {
    console.log("Grade: C");
} else {
    console.log("Grade: D");
}

Perulangan (Loop)

Digunakan untuk mengulang blok kode beberapa kali.

// For Loop
for (let i = 1; i <= 5; i++) {
    console.log("Iterasi ke-" + i);
}

// While Loop
let count = 1;
while (count <= 5) {
    console.log("Count: " + count);
    count++;
}

// For...of (untuk array)
let buah = ["Apel", "Jeruk", "Mangga"];
for (let item of buah) {
    console.log(item);
}

Fungsi (Function)

Fungsi adalah blok kode yang dapat dipanggil berkali-kali.

// Deklarasi fungsi
function sapa(nama) {
    return "Halo, " + nama + "!";
}

// Memanggil fungsi
let pesan = sapa("Budi");
console.log(pesan); // "Halo, Budi!"

// Arrow Function (ES6)
const tambah = (a, b) => a + b;
console.log(tambah(5, 3)); // 8

Array

Array adalah struktur data untuk menyimpan beberapa nilai dalam satu variabel.

// Membuat array
let buah = ["Apel", "Jeruk", "Mangga"];

// Mengakses elemen
console.log(buah[0]); // "Apel"

// Menambah elemen
buah.push("Pisang");

// Menghapus elemen terakhir
buah.pop();

// Method array penting
buah.length;           // Panjang array
buah.join(", ");      // Gabung jadi string
buah.slice(0, 2);     // Ambil sebagian array

Objek (Object)

Objek digunakan untuk menyimpan data dalam bentuk pasangan key-value.

// Membuat objek
let mahasiswa = {
    nama: "Budi",
    umur: 20,
    jurusan: "Informatika",
    aktif: true
};

// Mengakses properti
console.log(mahasiswa.nama);    // "Budi"
console.log(mahasiswa["umur"]); // 20

// Menambah properti
mahasiswa.semester = 4;

// Method dalam objek
let mobil = {
    merk: "Toyota",
    warna: "Merah",
    jalan: function() {
        return this.merk + " sedang berjalan";
    }
};

DOM Manipulation

DOM (Document Object Model) memungkinkan JavaScript memanipulasi HTML dan CSS.

// Mengambil elemen
let element = document.getElementById("myId");
let elements = document.getElementsByClassName("myClass");
let element = document.querySelector(".myClass");

// Mengubah konten
element.innerHTML = "Teks baru";
element.textContent = "Teks baru";

// Mengubah style
element.style.color = "red";
element.style.fontSize = "20px";

// Menambah/menghapus class
element.classList.add("active");
element.classList.remove("active");

// Event listener
element.addEventListener("click", function() {
    alert("Diklik!");
});

Event Handling

Event adalah aksi yang terjadi di halaman web, seperti klik, hover, atau input.

// Event onclick
<button onclick="alert('Hello!')">Klik Saya</button>

// Event listener
let tombol = document.getElementById("myButton");
tombol.addEventListener("click", function() {
    console.log("Button diklik!");
});

// Event-event umum
onclick      // Saat diklik
onmouseover  // Saat mouse di atas
onmouseout   // Saat mouse keluar
onchange     // Saat nilai berubah
onkeypress   // Saat tombol keyboard ditekan
onload       // Saat halaman selesai dimuat
