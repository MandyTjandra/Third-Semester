Materi Dasar CSS

Pelajari cara menghias dan menata halaman web Anda menjadi lebih menarik.

Apa itu CSS?

CSS adalah singkatan dari Cascading Style Sheets. Jika HTML adalah kerangka sebuah website, maka CSS adalah "kulit"-nya. CSS digunakan untuk mengatur tampilan visual elemen HTML, seperti warna, ukuran font, layout, dan banyak lagi.

Cara Menambahkan CSS

Ada tiga cara untuk menambahkan CSS ke dokumen HTML:

1. External CSS (Terbaik) ⭐

Menulis CSS di file terpisah dan menghubungkannya ke HTML.

<!-- Di dalam <head> HTML -->
<link rel="stylesheet" href="style.css">

2. Internal CSS

Menulis CSS di dalam tag <style> di HTML.

<style>
  p {
    color: blue;
  }
</style>

3. Inline CSS

Menambahkan style langsung ke elemen HTML.

<p style="color: blue;">Teks biru</p>

Sintaks dan Selector

Aturan CSS terdiri dari selector dan blok deklarasi. Selector menargetkan elemen HTML yang ingin Anda gayakan.

/* Selector elemen */
p {
  color: blue;
  font-size: 16px;
}

/* Selector class */
.kotak {
  background-color: yellow;
}

/* Selector ID */
#header {
  padding: 20px;
}

/* Selector multiple */
h1, h2, h3 {
  font-family: Arial, sans-serif;
}

Jenis-jenis Selector:

  • p → Semua elemen <p>
  • .classname → Semua elemen dengan class="classname"
  • #idname → Elemen dengan id="idname"
  • * → Semua elemen
  • div p → Semua <p> di dalam <div>

Warna (Colors)

Ada beberapa cara untuk menentukan warna di CSS:

/* Nama warna */
color: red;

/* Hexadecimal */
color: #0056b3;

/* RGB */
color: rgb(0, 86, 179);

/* RGBA (dengan transparansi) */
color: rgba(0, 86, 179, 0.5);

Hasil Tampilan:

Properti Font & Teks

CSS menyediakan banyak properti untuk mengatur tampilan teks.

.teks-styling {
  color: #0056b3;                 /* Warna teks */
  font-size: 20px;                /* Ukuran font */
  font-family: Georgia, serif;    /* Tipe font */
  font-weight: bold;              /* Ketebalan font */
  text-align: center;             /* Perataan teks */
  text-decoration: underline;     /* Garis bawah */
  text-transform: uppercase;      /* Huruf kapital */
  line-height: 1.5;               /* Jarak antar baris */
}

Konsep Box Model

Setiap elemen HTML dapat dianggap sebagai sebuah "kotak". Box Model menjelaskan bagaimana kotak ini disusun, yang terdiri dari: Content, Padding, Border, dan Margin. 

.kotak-contoh {
  width: 200px;                   /* Lebar konten */
  padding: 20px;                  /* Jarak di dalam border */
  border: 5px solid red;          /* Garis tepi */
  margin: 30px;                   /* Jarak di luar border */
  background-color: lightblue;    /* Warna latar */
}

Content (200px)
Area merah muda adalah visualisasi dari margin (30px).

Display Property

Properti display mengatur bagaimana elemen ditampilkan di halaman.

/* Block: Mengambil lebar penuh */
display: block;

/* Inline: Hanya mengambil ruang yang diperlukan */
display: inline;

/* Inline-block: Gabungan inline & block */
display: inline-block;

/* None: Menyembunyikan elemen */
display: none;

Flexbox Layout

Flexbox adalah sistem layout yang powerful untuk mengatur elemen secara fleksibel.

.container {
  display: flex;
  justify-content: center;  /* Horizontal alignment */
  align-items: center;      /* Vertical alignment */
  gap: 10px;                /* Jarak antar item */
}

/* Contoh penggunaan */
.flex-container {
  display: flex;
  justify-content: space-between;
}

Position Property

Properti position mengatur posisi elemen di halaman.

/* Static: Posisi default */
position: static;

/* Relative: Relatif terhadap posisi normal */
position: relative;
top: 10px;
left: 20px;

/* Absolute: Relatif terhadap parent */
position: absolute;

/* Fixed: Tetap di layar saat scroll */
position: fixed;

/* Sticky: Gabungan relative & fixed */
position: sticky;
top: 0;

Responsive Design (Media Queries)

Media queries membuat website responsif di berbagai ukuran layar.

/* Desktop first approach */
.container {
  width: 1000px;
}

/* Tablet */
@media (max-width: 768px) {
  .container {
    width: 100%;
    padding: 20px;
  }
}

/* Mobile */
@media (max-width: 480px) {
  .container {
    padding: 10px;
  }
  
  .text {
    font-size: 14px;
  }
}

Transition & Animation

Membuat efek transisi dan animasi pada elemen.

/* Transition */
.button {
  background-color: blue;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: red;
}

/* Animation */
@keyframes slideIn {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

.element {
  animation: slideIn 1s ease-in-out;
}
