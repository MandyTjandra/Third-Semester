Materi Dasar HTML

Pelajari fondasi utama dalam membangun struktur halaman web.

Struktur Dasar Dokumen

Setiap file HTML memiliki struktur dasar yang wajib ada. Ini memberitahu browser bahwa ini adalah dokumen HTML dan mendefinisikan bagian <head> (info halaman) dan <body> (konten halaman). 

<!DOCTYPE html>
<html lang="id">
<head>
    <title>Judul Halaman Web</title>
</head>
<body>
    <!-- Konten website ditulis di sini -->
</body>
</html>

Heading & Paragraf

Gunakan <h1> sampai <h6> untuk judul dan <p> untuk teks biasa. 

<h1>Ini adalah Judul Utama</h1>
<h2>Ini Sub-Judul</h2>
<p>Ini adalah sebuah paragraf teks biasa.</p>

Hasil Tampilan:

Ini adalah Judul Utama

Ini Sub-Judul

Ini adalah sebuah paragraf teks biasa.

Link (Hyperlink)

Gunakan tag <a> dengan atribut href untuk membuat link.

<a href="https://www.google.com">Kunjungi Google</a>

Hasil Tampilan:

Kunjungi Google

Gambar

Gunakan tag <img> untuk menampilkan gambar dan tag <p> dengan class khusus untuk keterangan.

<img src="Logo.png" alt="Gambar Contoh">
<p class="image-caption">Yes, itu logoku</p>

List (Daftar)

Gunakan <ul> untuk daftar tak berurutan (bullet) dan <ol> untuk daftar berurutan (angka).

<h4>Daftar Buah:</h4>
<ul>
    <li>Apel</li>
    <li>Jeruk</li>
</ul>

<h4>Langkah-langkah:</h4>
<ol>
    <li>Langkah Pertama</li>
    <li>Langkah Kedua</li>
</ol>

Hasil Tampilan:

Daftar Buah:

  • Apel
  • Jeruk

Langkah-langkah:

  1. Langkah Pertama
  2. Langkah Kedua

Tabel

Gunakan <table> untuk membuat tabel, <tr> untuk baris, <th> untuk header, dan <td> untuk sel data. 

<table>
    <tr>
        <th>Nama</th>
        <th>Kota</th>
    </tr>
    <tr>
        <td>Budi</td>
        <td>Jakarta</td>
    </tr>
</table>

Hasil Tampilan:

Nama Kota
Budi Jakarta

Form

Form digunakan untuk mengumpulkan input dari pengguna. Dibuat dengan tag <form> dan berbagai jenis elemen <input>. 

<form>
    <label for="nama">Nama:</label><br>
    <input type="text" id="nama" name="nama"><br><br>
    
    <label for="email">Email:</label><br>
    <input type="email" id="email" name="email"><br><br>
    
    <input type="submit" value="Kirim">
</form>
