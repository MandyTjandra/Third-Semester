## ðŸ† Featured Projects

### 1. Catch Him! (Turn-Based Strategy)
A grid-based strategy game where you play as the Police trying to catch Robbers before they escape. It features pathfinding logic and turn-based movement.

* **Mechanics:** Grid movement, obstacle destruction, AI pathfinding.
* **Controls:**
    * `Arrow Keys`: Move Police
    * `1-5`: Set step distance
    * `Space`: Use Bomb (Destroy walls)

### 2. Neon Shooter (Top-Down Shooter)
An arcade shooter featuring glowing vector graphics and particle effects. Survive as long as you can against waves of enemies.

* **Mechanics:** 360Â° shooting, particle systems, health bars, screen shake.
* **Controls:**
    * `WASD`: Move Player
    * `Mouse`: Aim
    * `Left Click`: Shoot

### 3. Avoid the Falling Blocks
A fast-paced survival game where the difficulty scales over time.

* **Mechanics:** Collision detection, increasing speed difficulty curve.
* **Controls:**
    * `Arrow Keys`: Move Player (Up, Down, Left, Right)
    * `R`: Restart game

### 4. Colorful Fading Circles
An interactive visualizer that spawns fading circles on click.
